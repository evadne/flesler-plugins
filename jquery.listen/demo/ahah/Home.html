This plugin brings a clean, light solution, to websites with dynamically generated content, or full of event bindings. Instead of bound, handlers for events, are registered along with matching selectors. And they'll still work for newly added content. This is achieved, using event delegation, so the plugin will only work for events that bubble.

The plugin supports these kind of selectors:

  • #id
  • nodeName
  • .class
  • nodeName.class

I plan to add more support, but perfomance and scalability are very important too. These simple selectors give enough functionality for many cases.
